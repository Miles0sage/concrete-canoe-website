# File Organization Guide

## 📁 Folder Structure

```
concrete-canoe/
├── documents/          # Place all PDF files here
├── images/
│   ├── team/          # Team member photos
│   ├── gallery/       # Gallery/project photos
│   └── project/       # Project-specific images (canoe renders, maps, etc.)
├── css/
├── js/
└── *.html
```

## 🔗 File Naming Best Practices

1. Use **lowercase** letters
2. Use **hyphens** instead of spaces
3. Keep names **descriptive** but **concise**
4. Use common formats: `.pdf`, `.jpg`, `.png`

## ✅ After Adding Files

Once you've added your files, the website will automatically link to them. No code changes needed!

The paths are already set up in:
- `documents.html` - for PDFs
- `index.html` - for team photos
- `project-info.html` - for project images
- `design.html` - for design images
